# r-help

The main R mailing list

Automated mbox archive. Updated daily via GitHub Actions.

## Structure

- `raw/` - Monthly .mbox files downloaded from [https://stat.ethz.ch/pipermail/r-help/](https://stat.ethz.ch/pipermail/r-help/)
- `processed/` - Structured JSON output from rmail-parser
- `meta.json` - List statistics

## Viewer
